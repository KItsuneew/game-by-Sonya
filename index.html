Краткий экскурс по игре

    Есть два основных режима игры
  • Gachiмученник
  • Борщевар

В режиме Gachiмученник отсутствует время на игру и ходы, которые будут как-то ограничивать геймплей

Также в данном режиме ограниченное количество карточек, их 16. Больше не будет, то есть это режим чисто поиграть просто так

В режиме Борщевар уже будет два выбора режима это: По времени и Время + ходы

В первом режиме будет ограничение по времени в зависимости от выбранной сложности.

Во втором режиме помимо огранения по времени будет ограничение по количеству ходов

Существует три сложности игры:

Easy (16 карточек): при выборе данной сложности в режиме По времени дается 64 секунды на игру / При выборе Время + Ходы вдобавок дается 24 хода Medium (36 карточек): при выборе данной сложности в режиме По времени дается 144 секунды на игру / При выборе Время + Ходы вдобавок дается 54 хода Hard (64 карточки): при выборе данной сложности в режиме По времени дается 256 секунд на игру / При выборе Время + Ходы вдобавок дается 96 ходов

Есть возможность ставить таймер на паузу во время игры и также его возобновлять, во время паузы не будет возможности нажимать на карточки, они будут неактивными, можете не пробовать даже)

Игра запускается после нажатия на кнопку старта Таймера
